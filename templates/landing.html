Real questions, real answers.

Do I need a fancy PC?

For the full desktop experience: 16 GB RAM and any GPU made in the last 5 years runs Llama 3.1 8B + Kokoro fine. For mobile-only use: any flagship phone from the last 3 years works (Pixel 7+, Galaxy S22+, iPhone 13+). The mobile PWA uses Llama 3.2 1B which fits in 2 GB RAM.

Does this actually work on a phone with no PC running?

Yes. Pick "This device" as the AI engine in Settings. Llama 3.2 1B downloads once (~1 GB, cached forever). After that: scripts generate via WebGPU, narration uses your phone's built-in TTS, music comes from the bundled pack, and ffmpeg.wasm renders the MP4 in your browser. Airplane mode safe.

What's free vs Pro?

Free tier: 5 video renders per month, all local AI features, project library. Pro: unlimited renders, no watermark, publish scheduler, SEO Finder, Optimize Library (vidIQ-aware repackaging), channel analytics ingest, brand presets, multi-channel.

What does Founding Lifetime get me?

Everything in Creator Pro, forever. Pay once, no subscription. Limited to the first 200 buyers. Locked-in price even if monthly tiers go up later. After the 200 are sold, the founding deal is gone.

Is my data sent anywhere?

No. Scripts, audio, video, channel analytics — all stay on your device. License keys are HMAC-signed offline. The only external traffic is optional: YouTube publishing if you connect it, Pexels API if you add a key for stock B-roll, anonymous error telemetry if you opt in.

Can I use this commercially?

Yes. Any video you make is yours. The bundled music pack is procedurally generated and public-domain. Generated narration belongs to you. Generated visuals belong to you. Phantomline charges for the tool, not for what you make with it.

Does it run on Mac and Linux?

The desktop app is Python — runs anywhere Python runs. Some font paths are Windows-tuned but auto-fall-back to Mac and Linux system fonts. The PWA runs in any modern browser on any OS.

Will my license expire?

Founding Lifetime: never. Creator Pro and Studio yearly: one year from purchase, renews. Monthly: 31 days, renews. Free tier: never expires. License validation is offline — losing internet doesn't lock you out.
