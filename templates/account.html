Easiest: double-click the launcher

  • Windows: double-click start-phantomline.bat in your Phantomline folder.
  • macOS: double-click start-phantomline.command (right-click → Open the first time, since it’s unsigned).
  • Linux: run ./start-phantomline.sh from the Phantomline folder.

Or via terminal:

cd path/to/phantomline
python server.py

Don’t have it installed yet? Install Phantomline desktop → takes ~3 minutes.
